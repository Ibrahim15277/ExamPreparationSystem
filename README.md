Быстрый запуск в Docker

Приложение полностью контейнеризировано.
Трнбования:
- Установленный [Docker Desktop]

Запуск одной командой
Находясь в корневой директории проекта (где расположен файл `docker-compose.yml`), выполните в терминале:

``` bash
docker-compose up -d --build
```

После успешного развертывания приложение будет доступно в браузере по адресу:
http://localhost:5000
